Instala Remote Linux Player

Para usar el control, añádela a tu pantalla de inicio. Se abrirá como una app, sin la barra del navegador.

iPhone / iPad (Safari)

  1. Toca Compartir (el cuadro con la flecha).
  2. Elige "Añadir a pantalla de inicio".
  3. Toca Añadir y abre desde el nuevo icono.

Android (Chrome)

  1. Pulsa el menú (tres puntos, arriba a la derecha).
  2. Elige "Instalar app" o "Añadir a pantalla de inicio".
  3. Confirma y abre desde el nuevo icono.

¿Ya la instalaste? Ábrela desde el icono, no desde el navegador.

Tutoriales
Usar el control

El panel central funciona como un touchpad. Toca para hacer clic, desliza con dos dedos para hacer scroll (o usa la barra lateral derecha).

Modo Navegación

Toca el botón con forma de cruceta. Ahora, al deslizar enviarás flechas (Arriba/Abajo/Izquierda/Derecha), un toque corto es Enter y mantener es Esc. Ideal para Bigscreen.

Emparejar un teléfono

En la TV, ve a LRP y anota el código. Ingresa el código en tu móvil para emparejarlos en la red local.
