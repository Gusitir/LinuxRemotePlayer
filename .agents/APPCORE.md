# DIRECTORIES
- /backend/: FastAPI server, Python logic.
- /frontend/: PWA static files.
- /scripts/: Bash setup.

# CRITICAL FILES
- /backend/main.py: FastAPI entrypoint & WS router.
- /backend/input_emulator.py: evdev UInput virtual gamepad wrapper.
- /backend/requirements.txt: Dependencies.
- /frontend/index.html: PWA UI base (Tailwind CDN).
- /frontend/app.js: WS client connection and payload sender.
